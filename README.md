TaskMate

TaskMate ist eine einfache Java-Konsolenanwendung zum Verwalten von Aufgaben.
Aufgaben können erstellt, angezeigt, erledigt und gelöscht werden.
Die Speicherung erfolgt in einer JSON-Datei.

Funktionen
	•	Aufgaben hinzufügen (Titel, Datum, Priorität)
	•	Aufgabenliste anzeigen
	•	Aufgaben als erledigt markieren
	•	Aufgaben löschen
	•	Speicherung in tasks.json
  
Technologie
	•	Java 17
	•	Maven
	•	Gson (JSON Serialisierung)
	•	CLI-basierte Bedienung

Ziel des Projekts

Lernen und Anwenden von Java, OOP, Maven und JSON in einer vollständigen kleinen Anwendung.
